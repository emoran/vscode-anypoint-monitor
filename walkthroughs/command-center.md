## Application Command Center

Your **unified control room** for any CloudHub 1.0, 2.0, or Hybrid application.

### Dashboard Sections:

- **Health Score** — Intelligent health calculation with breakdown
- **AI Insights** — Automated recommendations based on performance data
- **Performance Metrics** — Real-time CPU, memory, and network charts
- **Recent Logs** — Last 50 logs with quick filtering
- **Deployment History** — Track recent deployments and changes
- **Application Overview** — Runtime version, workers, replicas, region info

### Quick Actions:

- Restart application
- Open Real-Time Logs
- View Application Diagram
- Export dashboard data

> **Tip:** The health score updates in real time — use it as your primary indicator for application wellness.
