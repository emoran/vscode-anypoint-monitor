## Real-Time Log Streaming

Stream live application logs for both **CloudHub 1.0** and **CloudHub 2.0** applications with powerful filtering and export capabilities.

### Features:

- **Live Tailing** — Logs auto-refresh in real time
- **Level Filtering** — Filter by INFO, WARN, ERROR, or DEBUG
- **Content Search** — Search log messages by keyword
- **Thread Filtering** — Isolate logs from specific threads
- **Color-Coded Levels** — Quickly spot errors and warnings
- **Multi-Format Export** — Download as JSON, CSV, or TXT

### Supported Runtimes:

- **CloudHub 1.0** — Uses `/cloudhub/api/v2/applications/{name}/logs`
- **CloudHub 2.0** — Uses deployment specs API for log retrieval

> **Tip:** Use `AM: Application Command Center` to access logs alongside metrics and AI insights in one unified view.
